
Isomorphic React Redux application
------------------------------------
Initial Boilerplate: https://github.com/WebbyLab/itsquiz-wall

## Installation (development)

1. Inside folder etc create ``no-share.json`` and insert:
 {
  "googleClientID": "",
  "googleClientSecret": "",
  "facebookClientID": "",
  "facebookClientSecret": ""
}
2. ```npm install``` 
3. ```npm run dev```
